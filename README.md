## Depedencies

This project, use the React.js, Firebase, and NodeJS technology.

![RFN](https://i.ibb.co/h2FFgRT/ig.png)

First, you should install

-   [ReactJS](https://reactjs.org/)
-   [Node](https://nodejs.org/en/)

## Configuring App

```
$ cd (the directroty)
$ npm install
```
after finish, you can run

```
$ npm start
```
